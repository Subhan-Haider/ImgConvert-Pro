# Security Policy

The following versions of **Image Converter Pro** are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Our Commitment to Privacy

**Image Converter Pro** is built with a "Privacy First" architecture. 
- **100% Local Processing:** Every image conversion, compression, and manipulation happens directly in your browser or local environment.
- **No Server Uploads:** We do not maintain any servers for image processing. Your data never leaves your device.
- **Zero Tracking:** We do not collect personal identifyable information (PII) or usage telemetry of your files.

## Reporting a Vulnerability

We take the security of our users seriously. If you discover a security vulnerability within this project, please follow these steps:

1. **Do Not Open a Public Issue:** To prevent exploitation, please do not disclose vulnerabilities in public GitHub issues or discussions.
2. **Email Disclosure:** Please report vulnerabilities by emailing [subhanhaider.work@gmail.com](mailto:subhanhaider.work@gmail.com).
3. **Details to Include:**
   - A detailed description of the vulnerability.
   - Steps to reproduce (Proof of Concept).
   - Potential impact if exploited.

### Response Time
We aim to acknowledge your report within **48 hours** and provide a timeline for a fix within **5 business days**.

## Secure Practices for Users

- **Official Sources:** Only download the Chrome Extension from the official Web Store or the Desktop application from this official repository.
- **Updates:** Ensure you are using the latest version of your web browser or the latest release of the desktop app to benefit from the latest security patches.
